음악 스타일 반영 퀄리티 평가
(Music Stylization Quality Evaluation)



[Kor] 본 설문은 스타일 오디오를 활용하여, 컨텐츠 오디오의 스타일이 얼마나 변화되었는지를 평가하기 위한 사용자 연구입니다. 아래에 제시된 컨텐츠 오디오와 스타일 오디오를 먼저 들으신 후, 여러 개의 결과 아웃풋 오디오 중에서 스타일 오디오의 특징을 얼마나 잘 반영하고 있는지에 따라 각 아웃풋에 점수를 매겨주시기 바랍니다.

[Eng] This user study aims to evaluate how effectively the style of a reference audio clip is reflected in the stylized outputs. Please listen to the given content audio and style reference audio first. Then, for each stylized output, rate how well it captures the style characteristics of the reference audio.
